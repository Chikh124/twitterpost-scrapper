# Інструкція з налаштування проекту

## Крок 1: Встановлення Node.js

**Node.js не встановлено на вашому комп'ютері!**

1. Відкрийте браузер і перейдіть на: https://nodejs.org/
2. Завантажте **LTS версію** (рекомендовано v18 або новіше)
3. Запустіть завантажений установщик
4. Дотримуйтесь інструкцій установки (залиште всі налаштування за замовчуванням)
5. Перезапустіть термінал/командний рядок після встановлення

## Крок 2: Перевірка встановлення

Після встановлення Node.js, відкрийте новий термінал і виконайте:

```bash
node --version
npm --version
```

Обидві команди повинні показати версії.

## Крок 3: Встановлення залежностей проекту

Перейдіть у папку проекту і виконайте:

```bash
npm install
```

**Примітка:** Перше встановлення може зайняти 5-10 хвилин, оскільки Puppeteer завантажує Chromium (~300MB). Це нормально!

**Якщо ви хочете використовувати скрапінг для старих твітів (>7 днів):**
- Puppeteer потребує Chrome браузер
- Варіанти встановлення:
  1. Встановіть Chrome вручну: https://www.google.com/chrome/
  2. Або запустіть `install-chrome.bat` (подвійний клік на файл)
  3. Або в CMD (не PowerShell): `npx puppeteer browsers install chrome`
  4. Або: `npm run install-chrome`

## Крок 4: Налаштування API ключів

1. Відкрийте файл `.env` у папці проекту
2. Заповніть ваші Twitter API credentials:
   - Або `TWITTER_BEARER_TOKEN` (простіше, але обмежений функціонал)
   - Або OAuth 1.0a ключі (для повного доступу, включаючи лайки)

## Крок 5: Запуск проекту

```bash
npm start "https://x.com/username/status/TWEET_ID"
```

Або для старіших твітів (більше 7 днів):

```bash
npm start "https://x.com/username/status/TWEET_ID" --scrape
```

---

## Швидкий старт (після встановлення Node.js)

Якщо Node.js вже встановлено, просто виконайте:

```bash
npm install
```

Потім налаштуйте `.env` файл і запускайте проект!
